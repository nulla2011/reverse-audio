A Chrome browser extension designed to **record** and **reverse** any audio on webpage.

It supports Chrome and Chrome-based browsers.

# Installation

You can install the extension manually by loading it as an unpacked extension:

1. Download the latest .zip from Releases.

2. Extract the downloaded file.

3. Open Chrome and visit: chrome://extensions/

4. Enable Developer mode (top-right corner).

5. Click Load unpacked.

6. Select the extracted folder.

7. Pin the extention.

# Development

## Setup

```bash
$ pnpm install
```

## Start dev server

```bash
$ pnpm dev
```

## Build

```bash
$ pnpm build
```
